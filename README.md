# Real Estate Admin Dashboard

A modern **React-based admin dashboard** developed to manage and monitor the entire **Real Estate System** (AqarGo), which includes both the User and Office mobile applications built with Flutter.  
The dashboard provides administrators with full control over properties, users, offices, and transactions through a clean and responsive interface.

**Behance Project:** View the full project on [Behance](https://www.behance.net/gallery/236452627/AqarGo_App)

---

##  Key Features
- Manage and monitor **users, offices, and properties** in real time.  
- **Approve or reject** property requests submitted by users.  
- Track **subscriptions, payments, and points** used by offices.  
- View detailed analytics and performance insights.  
- Responsive layout compatible with both **desktop and tablet screens**.  
- Built with **Bootstrap 5** for a sleek and consistent UI.  
- Integrated with the backend using **Axios** for secure API communication.  

---

##  Tech Stack / Libraries
- **Framework:** React (v19)  
- **Routing:** React Router DOM (v7)  
- **UI & Styling:** Bootstrap 5, Remix Icons, Bootstrap Icons  
- **Networking:** Axios  
- **Testing:** React Testing Library, Jest DOM  
- **Performance Monitoring:** Web Vitals

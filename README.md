# Real Estate Admin Dashboard

A **React-based admin dashboard** built for the **AqarGo Real Estate System**, allowing administrators to efficiently manage and monitor system operations.  
The dashboard provides essential tools for handling users, offices, properties, and financial data with a simple and intuitive interface.

**Behance Project:** View the full project on [Behance](https://www.behance.net/)

## üß© Key Features
- Manage **users, offices, and properties** in one place.  
- View system **statistics and earnings reports**.  
- **Approve, reject, or delete** property listings and user requests.  
- Track **subscriptions, payments, and point balances**.  
- Responsive layout designed for **desktop use** with clear navigation.  
- Built with **Bootstrap 5** for consistency and performance.
